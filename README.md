# WCO Tag ID Analysis

This project analyzes the WCO dataset with Tag ID as of 06/06/24. It focuses on interpreting ecological health indicators using various quantitative methods. Data is collected from the WCO acoustic reciever array in the Channel Islands National Marine sanctuary and transmitters are attached to various marine species.

## Project Structure

- **data/**: Contains the raw data files.
- **scripts/**: Contains the analysis scripts.
- **outputs/**: Contains the generated output files (e.g., plots, results).
- **docs/**: Contains the documentation and reports.
